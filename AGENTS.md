# AGENTS.md

## 專案定位
本專案是一套 Python 入門電子書，目標讀者為初學者、高中生、大學生、非資訊背景自學者。

## 寫作範圍
內容只寫到 Function，不進入 class、module 等進階主題。

## 寫作語氣
- 白話
- 教學導向
- 適合第一次學 Python 的讀者
- 每段盡量短
- 優先用台灣常用術語

## 章節格式規範
每章固定包含：
1. 本章重點
2. 概念說明
3. 範例程式
4. 程式解說
5. 常見錯誤
6. 練習題
7. 小結

## 程式碼規範
- 使用 Python 3 語法
- 範例必須可執行
- 變數命名清楚
- 避免一次出現太多新觀念
- 每個範例聚焦一個核心概念

## Markdown 規範
- 使用 `#`、`##`、`###` 標題階層
- 程式碼使用 fenced code block
- Python 程式碼標註 `python`
- 非必要時不要直接硬寫 HTML 內容

## 不要做的事
- 不要跳級解釋太多進階語法
- 不要混入過多理論
- 不要使用中國大陸用語
- 不要在未說明前使用 class、decorator、generator

## 修改原則
- 優先維持整本書風格一致
- 新增內容時不要破壞既有章節編號
- 若補充內容，優先放在對應章節內，不新增無關章節
