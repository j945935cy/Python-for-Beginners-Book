# 第 2 章 變數與資料型態

## 本章重點
- 認識變數
- 了解常見資料型態
- 學會儲存與查看資料

## 概念說明
變數就像一個有名字的盒子，可以把資料放進去，之後再取出來使用。

## 範例程式
```python
name = "Amy"
age = 18
print(name)
print(age)
```

## 程式解說
`name` 儲存文字，`age` 儲存整數，兩者都是變數。

## 常見錯誤
- 變數名稱中有空白
- 把文字忘記加引號

## 練習題
1. 建立一個 `city` 變數。
2. 建立一個 `score` 變數並輸出。
3. 試著修改變數內容後再印出。

## 練習題參考解答與解析
### 第 1 題
```python
city = "Taipei"
print(city)
```
解析：先建立變數，再把城市名稱放進去即可。

### 第 2 題
```python
score = 100
print(score)
```
解析：這題是練習把數字存進變數，並用 `print()` 看結果。

### 第 3 題
```python
name = "Amy"
print(name)

name = "Jack"
print(name)
```
解析：同一個變數可以重新指定新的值，後面印出的結果就會跟著改變。

## 小結
學會變數後，就能讓程式記住資料並重複使用。
