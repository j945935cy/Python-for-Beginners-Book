# 第 3 章 輸入與輸出

## 本章重點
- 了解什麼是輸入與輸出
- 學會使用 `print()` 顯示文字、數字與變數
- 學會使用 `input()` 接收使用者輸入
- 知道輸入的資料通常會先以字串形式存起來

## 概念說明
前面幾章我們已經學過變數，也看過程式可以把結果顯示在畫面上。從這一章開始，我們會讓程式更有互動感。

所謂的**輸出**，就是程式把訊息顯示給使用者看。例如顯示歡迎文字、計算結果、提醒訊息，這些都屬於輸出。

所謂的**輸入**，就是讓使用者把資料送進程式裡。例如輸入名字、年齡、分數，讓程式拿來做後續處理。

你可以把它想成一個很簡單的流程：

1. 使用者先輸入資料
2. 程式接收資料並存進變數
3. 程式把結果顯示出來

這種流程看起來很基本，但幾乎所有互動式程式都會用到。像是登入系統、點餐系統、成績查詢、計算機，背後都離不開輸入與輸出。

在 Python 中，最常用的兩個工具就是：

- `print()`：負責輸出內容
- `input()`：負責接收輸入內容

只要先把這兩個工具學好，後面很多章節都會變得更好懂。

## 範例程式
### 範例 1：最基本的輸出
```python
print("歡迎來到 Python 世界！")
print("今天開始學寫程式")
```

### 範例 2：輸出變數內容
```python
student_name = "小美"
score = 95

print("學生姓名：", student_name)
print("考試分數：", score)
```

### 範例 3：接收名字並打招呼
```python
name = input("請輸入你的名字：")
print("你好，" + name)
```

### 範例 4：接收數字並進行簡單計算
```python
age_text = input("請輸入你的年齡：")
age = int(age_text)
next_year_age = age + 1

print("你今年", age, "歲")
print("明年你會", next_year_age, "歲")
```

## 程式解說
### 關於 `print()`
`print()` 的用途很直接，就是把內容顯示在畫面上。

在範例 1 中，我們把兩句文字直接放進 `print()` 裡，所以程式執行後就會一行一行顯示出來。

在範例 2 中，我們不是直接印死文字，而是先把資料放進變數，再把變數內容輸出。這樣的好處是，之後只要改變變數的值，輸出的結果也會跟著改變。

你也可以發現，`print()` 不只能印文字，也能印數字、變數，甚至一次印多個內容。對初學者來說，用逗號把資料分開是很方便的寫法。

### 關於 `input()`
`input()` 會先在畫面上顯示提示訊息，然後等待使用者輸入內容。

例如這一行：

```python
name = input("請輸入你的名字：")
```

它的意思是：
- 先顯示「請輸入你的名字：」
- 等待使用者輸入
- 把輸入結果存進 `name` 變數

接著我們再用 `print()` 把問候語顯示出來。這就是一個最基本的互動程式。

### 為什麼數字輸入還要轉型？
這是初學者很常卡住的地方。

雖然你在鍵盤輸入的是 18、20、35，看起來像數字，但 `input()` 接收到的內容，預設會先當成**文字**來處理。

所以如果你想拿它做加法、減法，通常要先轉成整數。這就是範例 4 中 `int(age_text)` 的用途。

```python
age_text = input("請輸入你的年齡：")
age = int(age_text)
```

第一行拿到的是文字。
第二行才把文字變成整數。

如果少了這一步，後面在做運算時就容易出錯。

## 常見錯誤
### 1. 忘記把輸入結果存進變數
錯誤寫法：
```python
input("請輸入你的名字：")
print(name)
```

這樣會有問題，因為你沒有把輸入結果存到 `name`，後面卻直接拿 `name` 來用。

正確寫法：
```python
name = input("請輸入你的名字：")
print(name)
```

### 2. 把輸入的數字直接拿去做運算
錯誤寫法：
```python
age = input("請輸入年齡：")
print(age + 1)
```

這裡的 `age` 還是文字，所以會出現型態不符合的錯誤。

正確寫法：
```python
age = int(input("請輸入年齡："))
print(age + 1)
```

### 3. 文字忘記加引號
錯誤寫法：
```python
print(歡迎來學 Python)
```

Python 會看不懂這串內容是什麼。

正確寫法：
```python
print("歡迎來學 Python")
```

## 練習題
1. 寫一支程式，先顯示一句歡迎訊息，再顯示你的名字。
2. 用 `input()` 讓使用者輸入姓名，接著輸出「歡迎你，某某某」。
3. 用 `input()` 讓使用者輸入年齡，並顯示「明年你會幾歲」。
4. 讓使用者輸入兩個整數，計算它們的總和並輸出結果。

## 練習題參考解答與解析
### 第 1 題
```python
print("歡迎來學 Python")
print("我的名字是 J5")
```
解析：這題主要是練習最基本的輸出，把兩句訊息依序顯示出來。

### 第 2 題
```python
name = input("請輸入你的姓名：")
print("歡迎你，" + name)
```
解析：用 `input()` 接收使用者輸入，再把變數內容串接到問候語中。

### 第 3 題
```python
age = int(input("請輸入你的年齡："))
print("明年你會", age + 1, "歲")
```
解析：因為年齡要做加法，所以要先把輸入文字轉成整數。

### 第 4 題
```python
num1 = int(input("請輸入第一個整數："))
num2 = int(input("請輸入第二個整數："))
print("總和是：", num1 + num2)
```
解析：這題把輸入、轉型、計算和輸出結合在一起，是很好的綜合練習。

## 小結
這一章學到的內容很重要，因為它是程式與使用者互動的第一步。

你已經學會：
- 用 `print()` 顯示訊息
- 用 `input()` 接收資料
- 把輸入內容存進變數
- 在需要計算時，先把文字轉成整數

只要把這些基本觀念練熟，後面在學條件判斷、迴圈和函式時，你就能更自然地寫出有互動的程式。
