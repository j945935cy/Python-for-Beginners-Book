## 概念說明
寫程式時，常常會遇到某些工作要重複做好幾次。

例如：
- 一直重複顯示歡迎訊息
- 多次計算兩個數字的總和
- 依照不同名字顯示不同問候語

如果每次都把相同的程式碼再寫一遍，不但麻煩，也很難維護。

這時候就可以使用 **函式（Function）**。

函式可以把一段有特定用途的程式整理起來，等到需要的時候再呼叫使用。

你可以把函式想成一台小機器：
- 給它資料
- 它幫你做事
- 再把結果顯示出來或回傳回來

函式的好處很多：
- 減少重複程式碼
- 讓程式更清楚
- 方便修改與重複使用

這也是你在本書中學到的一個很重要的整理能力。

## 範例程式
### 範例 1：最基本的函式
```python
def say_hello():
    print("歡迎來學 Python")

say_hello()
```

### 範例 2：有參數的函式
```python
def say_hello(name):
    print("你好，" + name)

say_hello("Amy")
say_hello("Jack")
```

### 範例 3：回傳結果的函式
```python
def add_numbers(x, y):
    result = x + y
    return result

total = add_numbers(10, 5)
print(total)
```

### 範例 4：搭配輸入使用
```python
def print_user_name(name):
    print("你的名字是：" + name)

user_name = input("請輸入名字：")
print_user_name(user_name)
```

## 程式解說
### 1. 如何定義函式
在 Python 中，定義函式要使用 `def`。

例如：

```python
def say_hello():
    print("歡迎來學 Python")
```

這裡的意思是：
- `def` 表示要定義一個函式
- `say_hello` 是函式名稱
- 後面的括號 `()` 是函式格式的一部分
- 冒號 `:` 後面接的是函式要做的事

注意，函式內的程式碼要記得縮排。

### 2. 定義後還要呼叫
這是非常重要的一點。

很多初學者以為函式定義完就會自動執行，其實不會。

你必須主動呼叫它：

```python
say_hello()
```

這樣程式才會真的去執行函式裡面的內容。

### 3. 什麼是參數
有時候同一個函式想處理不同資料，就可以使用**參數**。

例如：

```python
def say_hello(name):
    print("你好，" + name)
```

這裡的 `name` 就是參數。

它的意思是：
「當別人呼叫這個函式時，可以順便給我一個名字，我就用這個名字來打招呼。」

所以：

```python
say_hello("Amy")
say_hello("Jack")
```

雖然呼叫的是同一個函式，但會得到不同結果。

### 4. 什麼是回傳值
有些函式不是只做一件事然後印出來，而是會把結果送回去，讓外面繼續使用。

這時就會用到 `return`。

例如：

```python
def add_numbers(x, y):
    result = x + y
    return result
```

這個函式會把兩個數字相加，再把答案回傳出去。

外面就可以這樣接住它：

```python
total = add_numbers(10, 5)
print(total)
```

這樣做的好處是，計算結果可以繼續拿來做別的事情，而不只是當場印出來。

## 常見錯誤
### 1. 定義函式後忘記呼叫
例如：

```python
def say_hello():
    print("你好")
```

這樣只是在「定義」函式，還沒有真正執行。

如果你想看到結果，還要再加上：

```python
say_hello()
```

### 2. 參數數量不一致
如果函式定義時需要一個參數，呼叫時就要提供一個資料。

例如：

```python
def say_hello(name):
    print("你好，" + name)
```

如果你直接寫：

```python
say_hello()
```

就會出錯，因為少給了 `name`。

### 3. 縮排不正確
函式裡面的內容一定要縮排，不然 Python 會不知道哪些程式碼屬於函式本體。

## 練習題
1. 寫一個 `say_hi()` 函式，呼叫後印出一句問候語。
2. 寫一個 `show_name(name)` 函式，能接收名字並顯示歡迎訊息。
3. 寫一個 `add_numbers(a, b)` 函式，回傳兩數相加的結果。
4. 寫一個 `is_adult(age)` 函式，當年齡大於等於 18 時印出「成年」，否則印出「未成年」。
5. 想一想：如果同樣的程式碼會重複出現很多次，為什麼用函式會比較好？

## 練習題參考解答與解析
### 第 1 題
```python
def say_hi():
    print("嗨，歡迎你")

say_hi()
```
解析：先用 `def` 定義函式，再主動呼叫它。

### 第 2 題
```python
def show_name(name):
    print("歡迎你，" + name)

show_name("Amy")
```
解析：參數可以讓同一個函式處理不同資料。

### 第 3 題
```python
def add_numbers(a, b):
    return a + b

print(add_numbers(3, 5))
```
解析：如果希望把結果交回外面使用，就可以用 `return`。

### 第 4 題
```python
def is_adult(age):
    if age >= 18:
        print("成年")
    else:
        print("未成年")

is_adult(20)
```
解析：這題把函式和條件判斷結合在一起，是很好的總複習。

### 第 5 題
參考答案：因為函式可以減少重複程式碼，讓程式更整齊，也更容易修改和重複利用。

解析：這就是函式在程式設計中很重要的原因。
