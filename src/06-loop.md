# 第 6 章 迴圈

## 本章重點
- 認識什麼是重複執行
- 學會使用 `for` 迴圈
- 學會使用 `while` 迴圈
- 了解如何避免無限迴圈

## 概念說明
在寫程式時，我們常常會遇到一種情況：同一件事要做好幾次。

例如：
- 印出 1 到 10
- 重複顯示歡迎訊息 5 次
- 讓使用者一直輸入，直到輸入正確內容為止

如果每次都手動寫很多行 `print()`，不但麻煩，也很容易出錯。

這時候就需要用到**迴圈**。

迴圈的意思很簡單，就是讓一段程式重複執行。
你可以把它想成：「請這段程式再做一次、再做一次，直到符合停止條件為止。」

在 Python 裡，初學者最常接觸到兩種迴圈：

- `for`：適合已經知道要重複幾次的情況
- `while`：適合還不確定要重複幾次，要看條件是否成立的情況

這一章會帶你一步一步認識這兩種寫法。

## 範例程式
### 範例 1：使用 `for` 印出 0 到 4
```python
for i in range(5):
    print(i)
```

### 範例 2：使用 `for` 重複顯示訊息
```python
for i in range(3):
    print("歡迎學習 Python")
```

### 範例 3：使用 `while` 印出 1 到 5
```python
count = 1

while count <= 5:
    print(count)
    count = count + 1
```

### 範例 4：倒數計時
```python
number = 5

while number > 0:
    print(number)
    number = number - 1

print("時間到！")
```

## 程式解說
### 1. `for` 迴圈的基本概念
如果你已經知道某段程式要重複幾次，`for` 會很好用。

先看這段：

```python
for i in range(5):
    print(i)
```

這裡的意思是：
- `range(5)` 會產生 0、1、2、3、4
- 每次執行迴圈時，`i` 會依序拿到其中一個值
- 然後把 `i` 印出來

所以最後畫面上會看到：

```python
0
1
2
3
4
```

### 2. `range()` 的用法
`range()` 在 `for` 迴圈裡非常常見。

例如：
- `range(5)` 代表 0 到 4
- `range(1, 6)` 代表 1 到 5

很多初學者會以為 `range(5)` 會包含 5，但其實**不會包含最後那個數字**。

例如：

```python
for i in range(1, 6):
    print(i)
```

這樣才會印出 1 到 5。

### 3. `while` 迴圈的基本概念
如果你不知道程式要重複幾次，而是要看某個條件是否成立，就可以用 `while`。

例如：

```python
count = 1

while count <= 5:
    print(count)
    count = count + 1
```

這段程式的意思是：
- 只要 `count <= 5` 成立，就持續執行
- 每次執行完後，把 `count` 加 1
- 當 `count` 變成 6 時，條件不成立，迴圈就停止

### 4. 為什麼要更新變數
在 `while` 迴圈裡，很重要的一件事就是：**條件中的變數要改變**。

如果你忘記更新，程式就可能永遠不會停下來，這種情況叫做**無限迴圈**。

例如下面這種寫法就有問題：

```python
count = 1

while count <= 5:
    print(count)
```

因為 `count` 一直都是 1，條件永遠成立，所以程式會一直重複下去。

## 常見錯誤
### 1. 不清楚 `range()` 的範圍
初學者很常把 `range(5)` 誤解成 1 到 5。

其實它是從 0 開始，所以結果會是 0、1、2、3、4。

### 2. 忘記更新 `while` 內的變數
這是最常見的錯誤之一。

如果條件永遠成立，程式就會一直跑下去，不會自動停止。

### 3. 縮排不正確
`for` 和 `while` 底下要重複執行的程式，一定要有正確縮排。

例如：

```python
for i in range(3):
    print(i)
```

如果 `print(i)` 沒有縮排，就不會屬於迴圈的一部分。

## 練習題
1. 使用 `for` 迴圈印出 1 到 10。
2. 使用 `for` 迴圈重複輸出「我正在學 Python」5 次。
3. 使用 `while` 迴圈印出 1 到 5。
4. 寫一個倒數程式，從 10 倒數到 1，最後印出「結束」。
5. 想一想：什麼情況比較適合用 `for`？什麼情況比較適合用 `while`？

## 練習題參考解答與解析
### 第 1 題
```python
for i in range(1, 11):
    print(i)
```
解析：要印出 1 到 10，可以用 `range(1, 11)`。

### 第 2 題
```python
for i in range(5):
    print("我正在學 Python")
```
解析：這題重點是固定次數重複，適合用 `for`。

### 第 3 題
```python
count = 1
while count <= 5:
    print(count)
    count = count + 1
```
解析：使用 `while` 時別忘了更新變數，不然會變成無限迴圈。

### 第 4 題
```python
number = 10
while number >= 1:
    print(number)
    number = number - 1
print("結束")
```
解析：這題是 while 的經典練習，條件不成立時就會停止。

### 第 5 題
參考答案：
- 已知重複次數時，比較適合用 `for`
- 要依條件決定重複多久時，比較適合用 `while`

解析：這是兩種迴圈最重要的差別。

## 小結
這一章你已經學會 Python 中兩種重要的迴圈。

你現在知道：
- 迴圈可以讓程式重複執行工作
- `for` 適合用在已知次數的重複
- `while` 適合用在看條件是否成立的重複
- 使用 `while` 時要特別注意變數更新
- `range()` 不會包含最後一個數字

學會迴圈之後，你的程式就能更有效率地處理重複任務。
