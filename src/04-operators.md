# 第 4 章 運算子

## 本章重點
- 認識常見的數學運算子
- 學會比較兩個值的大小
- 了解布林值在判斷中的用途
- 為下一章的條件判斷打好基礎

## 概念說明
在日常生活中，我們常常會做很多比較與計算。

例如：
- 今天買東西總共多少錢？
- 考試分數有沒有及格？
- 年齡是不是超過 18 歲？

在 Python 裡，這些動作都要靠**運算子**來完成。

運算子可以把它想成「做事的符號」。
有些符號負責計算，像是加法、減法；有些符號負責比較，像是大於、小於、是否相等。

如果沒有運算子，程式就很難處理資料，也無法根據結果做判斷。

這一章會先從最常見的幾種開始學：

- 算術運算子：做加減乘除
- 比較運算子：比大小、比是否相等
- 邏輯概念：知道結果可能是對或錯

這些內容看起來不難，但非常重要。後面在寫 if 判斷、迴圈條件時，幾乎都會用到。

## 範例程式
### 範例 1：基本算術運算
```python
x = 10
y = 3

print("加法：", x + y)
print("減法：", x - y)
print("乘法：", x * y)
print("除法：", x / y)
```

### 範例 2：整除與餘數
```python
apples = 10
people = 3

print("每人可分到：", apples // people, "顆")
print("剩下：", apples % people, "顆")
```

### 範例 3：比較大小
```python
score = 75

print(score > 60)
print(score < 60)
print(score == 75)
```

### 範例 4：把比較結果用在生活情境
```python
age = 20
is_adult = age >= 18

print("是否成年：", is_adult)
```

## 程式解說
### 1. 算術運算子
先來看最常見的數學運算：

- `+`：加法
- `-`：減法
- `*`：乘法
- `/`：除法
- `//`：整除
- `%`：取餘數

範例 1 中的 `x + y`，意思就是把兩個數字加起來。

如果 `x = 10`、`y = 3`，那麼：
- `x + y` 會得到 13
- `x - y` 會得到 7
- `x * y` 會得到 30
- `x / y` 會得到 3.333333... 

這裡要特別注意，除法 `/` 的結果通常會是小數。

### 2. 什麼是整除與餘數
有時候我們不是只想知道除法結果，而是想知道：
- 一組東西平均分後，每人拿到幾個
- 最後會剩下多少

這時就會用到：
- `//`：整除，只保留整數部分
- `%`：餘數

例如 10 顆蘋果分給 3 個人：
- `10 // 3` 結果是 3
- `10 % 3` 結果是 1

這種寫法在分組、計算星期、判斷奇偶數時都很常見。

### 3. 比較運算子
除了計算，程式也很常需要做比較。

常見的比較運算子有：
- `>`：大於
- `<`：小於
- `>=`：大於等於
- `<=`：小於等於
- `==`：是否相等
- `!=`：是否不相等

比較後的結果只會有兩種：
- `True`：表示成立
- `False`：表示不成立

例如：

```python
print(8 > 5)
print(3 == 4)
```

第一行會得到 `True`，第二行會得到 `False`。

這樣的結果在下一章的 if 判斷會非常重要，因為 if 就是看條件是真還是假，再決定程式要怎麼做。

### 4. 把結果存進變數
你不一定每次都要直接印出結果，也可以先存進變數。

像這一行：

```python
is_adult = age >= 18
```

它的意思是把「是否成年」的判斷結果先存起來。
如果年齡大於等於 18，`is_adult` 就會是 `True`。

這樣的寫法會讓程式更容易閱讀。

## 常見錯誤
### 1. 把 `=` 和 `==` 混在一起
這是初學者最常犯的錯。

- `=` 是指定值給變數
- `==` 才是比較兩邊是否相等

錯誤觀念：
以為 `score = 60` 是在比較，其實不是，它是在把 60 放進 `score`。

### 2. 以為除法一定會得到整數
很多人看到 `10 / 2` 得到 5 沒問題，就以為除法都會是整數。
但其實 `/` 在 Python 裡通常會保留成小數格式。

如果你只想要整數部分，應該用 `//`。

### 3. 文字和數字直接相加
例如下面這種寫法就容易出錯：

```python
age = 18
print("我的年齡是：" + age)
```

因為前面是文字，後面是數字，型態不同。

比較簡單的寫法是：

```python
age = 18
print("我的年齡是：", age)
```

## 練習題
1. 建立兩個變數，分別存入 12 和 5，印出它們的加法、減法、乘法結果。
2. 試著計算 17 除以 4 的整除結果與餘數。
3. 建立一個 `score` 變數，判斷它是否大於等於 60。
4. 建立兩個數字，比較它們是否相等，並輸出結果。
5. 想一想：如果今天有 25 顆糖果，要分給 6 個人，應該用哪一個運算子來算每人拿幾顆？又要用哪一個運算子算剩下幾顆？

## 練習題參考解答與解析
### 第 1 題
```python
x = 12
y = 5
print(x + y)
print(x - y)
print(x * y)
```
解析：這題是練習最基本的算術運算子。

### 第 2 題
```python
print(17 // 4)
print(17 % 4)
```
解析：`//` 會算整除結果，`%` 會算餘數，所以答案分別是 4 和 1。

### 第 3 題
```python
score = 70
print(score >= 60)
```
解析：如果分數大於等於 60，結果就會是 `True`。

### 第 4 題
```python
a = 8
b = 8
print(a == b)
```
解析：`==` 是比較是否相等，不是指定值。

### 第 5 題
參考答案：
- 算每人分到幾顆要用 `//`
- 算剩下幾顆要用 `%`

解析：這題是在理解整除與餘數的生活應用。

- 比較結果會得到 `True` 或 `False`
